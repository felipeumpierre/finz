# German Tax Forms (Anlagen) Quick Reference

This reference maps tax situations to the correct ELSTER forms (Anlagen).

## Core Forms

| Form | Full Name | When Needed |
|------|-----------|-------------|
| **Mantelbogen / Hauptvordruck** | Main form | Always — personal data, bank details, filing status |
| **Anlage N** | Einkünfte aus nichtselbständiger Arbeit | Employment income — one per person per employer |
| **Anlage N-AUS** | Ausländische Einkünfte aus nichtselbständiger Arbeit | Foreign employment income |
| **Anlage Vorsorgeaufwand** | Vorsorgeaufwendungen | Insurance & pension contributions (always, if claiming beyond employer data) |
| **Anlage AV** | Altersvorsorgebeiträge (Riester) | Riester pension — to claim Sonderausgabenabzug |
| **Anlage Kind** | Kinder | One per child — Kindergeld, Kinderfreibetrag, childcare, school fees |
| **Anlage S** | Einkünfte aus selbständiger Arbeit | Freelance income (freie Berufe: artists, consultants, engineers, etc.) |
| **Anlage G** | Einkünfte aus Gewerbebetrieb | Business/trade income |
| **Anlage EÜR** | Einnahmenüberschussrechnung | Profit calculation for freelance/business (if no Bilanz) |
| **Anlage V** | Einkünfte aus Vermietung und Verpachtung | Rental income — one per property |
| **Anlage KAP** | Einkünfte aus Kapitalvermögen | Capital gains — only if claiming Günstigerprüfung or taxes not properly withheld |
| **Anlage R** | Renten und andere Leistungen | Pension income |
| **Anlage SO** | Sonstige Einkünfte | Other income (private sales within speculation period, alimony received, etc.) |
| **Anlage AUS** | Ausländische Einkünfte | Foreign income — for DBA credit/exemption method |
| **Anlage Haushaltsnahe** | Haushaltsnahe Aufwendungen | §35a — household services and craftsmen (sometimes integrated in Mantelbogen) |
| **Anlage Außergewöhnliche Belastungen** | Außergewöhnliche Belastungen | Medical expenses, disability, care costs |
| **Anlage Sonderausgaben** | Sonderausgaben | Donations, church tax, education costs (sometimes in Mantelbogen) |
| **Anlage Unterhalt** | Unterhaltsleistungen | Alimony/support payments to ex-spouse or relatives |

## Common Scenarios

**Married couple, both employed, one child, no other income:**
→ Mantelbogen + 2× Anlage N + Anlage Vorsorgeaufwand + Anlage Kind

**Same + one spouse has Riester:**
→ Add Anlage AV

**Same + freelance side income:**
→ Add Anlage S + Anlage EÜR

**Same + rental property:**
→ Add Anlage V (one per property)

## ELSTER Filing Notes

- ELSTER (elster.de) is free and the official electronic filing system
- First-time users need to register (takes a few days for activation code by mail)
- Most data from employers and insurance companies is pre-filled (vorausgefüllte Steuererklärung / VaSt)
- Deadline for electronic filing: same as paper (July 31 following year, or extended with Steuerberater)
